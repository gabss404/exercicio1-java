# 📚 Exercícios de Vetores em Java

Este repositório contém a resolução de 3 exercícios propostos no módulo 7, com foco em vetores, estruturas de repetição e manipulação de dados em Java.

---

## ✅ Exercício 1

### 📌 Descrição:

* Receber um vetor de números inteiros
* Ordenar o vetor (Bubble Sort)
* Receber um nome
* Contar a quantidade de vogais

### 💡 Conceitos utilizados:

* Vetores (`array`)
* Laço `for`
* Algoritmo de ordenação (Bubble Sort)
* Manipulação de `String` e `char`

---

## ✅ Exercício 2

### 📌 Descrição:

* Receber um vetor de números inteiros
* Percorrer o vetor:

  * Se o número for **par**, multiplicar por 2
  * Se for **ímpar**, elevar ao quadrado
* Exibir o novo vetor

### 💡 Conceitos utilizados:

* Vetores
* Operador `%` (módulo)
* Estrutura condicional `if/else`
* Função `Math.pow()`

---

## ✅ Exercício 3

### 📌 Descrição:

* Receber um nome do usuário
* Inverter o nome utilizando vetor
* Exibir o nome invertido

### 💡 Conceitos utilizados:

* Vetores de `char`
* Manipulação de `String`
* Troca de valores (swap)

---

## 🚀 Como executar

1. Compile os arquivos:

   ```bash
   javac Exercicio1.java Exercicio2.java Exercicio3.java
   ```

2. Execute cada exercício:

   ```bash
   java Exercicio1
   java Exercicio2
   java Exercicio3
   ```
